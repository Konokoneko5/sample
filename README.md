# Python Snake Game

A classic Snake game implemented in Python and designed to run directly in your web browser using Brython. Control the snake to eat food, grow longer, and avoid hitting the walls or its own tail!

## Technology Stack

*   **Python 3:** For the core game logic.
*   **Brython:** To enable Python execution in the browser, interacting with HTML DOM.
*   **HTML5:** For the game structure.
*   **CSS3:** For styling.

## How to Play

1.  **Clone or Download:** Get a copy of the game files onto your local machine.
2.  **Open the Game:** Navigate to the project directory and open the `index.html` file in a modern web browser (e.g., Chrome, Firefox, Edge, Safari).
3.  **Controls:**
    *   Use the **Arrow Keys** (`ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight`) on your keyboard to change the snake's direction.
4.  **Objective:**
    *   Guide the snake to eat the red food blocks that appear on the screen.
    *   Each piece of food eaten makes the snake grow longer and increases your score.
    *   The game ends if the snake hits the boundaries of the game area or collides with its own body.

## Project Structure

*   `index.html`: The main HTML file containing the game layout and the Brython Python script for game logic and rendering.
*   `static/css/style.css`: Contains the CSS rules for styling the game's appearance.
*   `README.md`: This file, providing information about the game.
